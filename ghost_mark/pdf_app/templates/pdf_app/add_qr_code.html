How It Works

The QR Code feature adds a small QR code to the bottom right corner of each page in your PDF. This QR code contains an encoded version of your email address that can be scanned later to verify the document's source.

Key Features:

  • Small, unobtrusive QR code placed in the corner of each page
  • Secure encoding of your email address for privacy
  • Works with any standard QR code scanner
  • Can be used for document tracking and verification
